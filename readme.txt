Explanation of the folder directories:
docs/ 		documentation for roc software
etc/  		operating system and boot commands for roc software
opt/roc  	binary executable and run time copies of configuration files
root/ roc 	software

Folder structures:
docs:
|-----html        web page documentation generated by doxygen
      |-----search
      |-----src   css source code to generate web page documentation with doxygen

root:                 roc source code
      |-----roc
            |-----config_generic   configuration settings for roc and gps
            |-----scripts_init.d   startup scripts for bootup of roc
            |-----src              roc and zephyr code for data recording and transfer

If the above folder structure is changed, make sure Doxyfile is updated accordingly under ~/docs/Doxyfile
In line 793, at the section for input tags, add both relative path to the ROC source codes (roc.c,roc.h,zephyr.c,zephyr.h) and 
doxygen setting source codes (customdoxygen.css,header.html,modules.dox,etc)
By default: INPUT = ./src ../root/roc/src
